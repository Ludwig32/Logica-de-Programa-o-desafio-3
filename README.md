# Logica-de-Programa-o-desafio-3
Este projeto possui dois códigos em JavaScript. O primeiro cria uma função que exibe a tabuada de um número de 1 a 10 usando um laço for. O segundo cria uma função que faz uma conversão simples de dólar para real com uma cotação fixa. Ambos demonstram o uso de funções, variáveis, loops e operações matemáticas básicas.
